# RZExtensions

Common extensions used for Apple platforms (e.g. `Combine`, `Foundation`, `Swift`, `SwiftUI`, `UIKit`, et cetera).

## Usage
Add the following URL as a Swift Package to the project:
```
https://github.com/rlziii/RZExtensions
```


## Minimum Targets
macOS: `11`
iOS/iPadOS: `14`
tvOS: `14`
watchOS: `7`
